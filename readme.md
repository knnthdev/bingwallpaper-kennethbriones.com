# Extensión de Wallpapers Diarios para GNOME en Pop!_OS

Esta extensión está diseñada exclusivamente para el entorno de escritorio GNOME en la versión actual de Pop!_OS.

## Funcionalidad

- Descarga automáticamente un wallpaper diario desde los servicios de Bing.
- Los wallpapers descargados se almacenan en una carpeta llamada `.cache` ubicada en tu directorio personal (`~/`).
- La extensión se ejecuta de forma automática, asegurando que siempre tengas un fondo de pantalla actualizado cada día.

## Requisitos

- Pop!_OS (versión actual) con entorno de escritorio GNOME.
- Conexión a Internet para descargar los wallpapers.

## Instalación

1. Clona este repositorio o descarga los archivos de la extensión.
2. Sigue las instrucciones específicas de instalación para extensiones de GNOME en Pop!_OS.
